cross-env解决windows上NODE_ENV的问题

发表于 2018-05-07 | 分类于 防坑指南

平时搭项目时本地起服务正常,但遇到用windows的同事down了项目后启动会报错,这里需要修改下package.json的scripts标签

windows不支持NODE_ENV=development的设置方式。
这个包能够提供一个设置环境变量的scripts，让你能够以unix方式设置环境变量，然后在windows上也能兼容运行。
