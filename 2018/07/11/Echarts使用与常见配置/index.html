Echarts 使用以及常用配置项操作

导语

Echarts，一个纯 Javascript 的图表库，可以流畅的运行在 PC 和移动设备上，兼容当前绝大部分浏览器（IE8/9/10/11，Chrome，Firefox，Safari等），底层依赖轻量级的 Canvas 类库 ZRender，提供直观，生动，可交互，可高度个性化定制的数据可视化图表。ECharts 3 中更是加入了更多丰富的交互功能以及更多的可视化效果，并且对移动端做了深度的优化。

认识Echarts

ECharts 提供了常规的折线图，柱状图，散点图，饼图，K线图，用于统计的盒形图，用于地理数据可视化的地图，热力图，线图，用于关系数据可视化的关系图，treemap，多维数据可视化的平行坐标，还有用于 BI 的漏斗图，仪表盘，并且支持图与图之间的混搭。

使用Echarts的优点

  • 兼容性强，移动、PC均达到高度自适应
  • 丰富的图表类型，日常能想到的均能有图表形式进行表达
  • 深度的交互式数据探索
  • 大数据量的表现
  • 多维数据的支持以及丰富的视觉编码手段
  • 动态数据
  • 绚丽的特效

配置项解析

1.Echarts组件;

1.1.图例组件:legend

1.2.视觉映射组件(地图):visualMap

1.3.数据区域缩放组件:dataZoom

1.4.时间线组件:timeline

2.Echarts两种事件种类:

2.1.用户鼠标操作点击:hover图标的图形时触发的行为事件

2.2.用户在使用了可以交互的组件后触发的行为事件

3.X轴:xAsix

3.1.type:A.value //数值轴(bar,line图) B.category //目类轴(pie图) C.time //时间轴 D.log //对数轴

3.2.boundaryGap:坐标轴两边是否留有空白,也就是顶格.false为不留空白,true为留空白

4.提示框:tooltip

4.1.trigger:A.item //数据项图形触发(pie图) B.axis //坐标轴触发(bar,line图)

5.工具栏:toolbox

5.1.feature //工具配置项

5.1.1.saveAsImage //保存为图片

5.1.2.restore //配置项还原

5.1.3.dataview //数据视图工具

5.1.4.dataZoom //数据区域缩放

5.1.5.magicType //动态类型切换
